Reproductor Multimedia Local

Rubén García November 4, 2025

Introduccion

El proyecto consiste en el desarrollo de un reproductor multimedia local con integración de calendario, cuyo objetivo principal es ofrecer una herramienta versátil para la gestión y reproducción de archivos de audio y video, junto con la organización de eventos o recordatorios relacionados. La aplicación busca combinar funcionalidad y diseño intuitivo, permitiendo al usuario reproducir su contenido multimedia de forma sencilla, al tiempo que puede planificar actividades o programar reproducciones según fechas específicas. Durante su desarrollo se aplicaron los principios de interfaz de usuario, usabilidad y experiencia de usuario (UX/UI), empleando tecnologías web estándar como HTML5, CSS y JavaScript vanilla.

Funcionalidades

El reproductor multimedia con calendario incorpora una serie de funcionalidades diseñadas para mejorar la experiencia del usuario y optimizar la gestión del contenido local. Entre las características más relevantes se incluyen:

  • Reproducción de archivos locales, permitiendo al usuario seleccionar y reproducir música o videos directamente desde su dispositivo.

  • Controles básicos de reproducción, como reproducir, pausar, detener, avanzar, retroceder y ajustar el volumen.

  • Lista de reproducción dinámica, que permite agregar, eliminar o reordenar archivos de manera sencilla.

  • Integración con calendario, posibilitando asociar eventos o recordatorios a fechas específicas, e incluso vincularlos con archivos multimedia concretos.

  • Interfaz intuitiva y adaptable, diseñada siguiendo principios de usabilidad y accesibilidad, con un diseño limpio y moderno.

  • Temas personalizables, incluyendo la opción de modo oscuro para mejorar la visualización en distintos entornos.

  • Persistencia de datos, de modo que las listas de reproducción y los eventos del calendario se mantengan entre sesiones mediante almacenamiento local.

Esta combinación de funciones busca ofrecer una aplicación completa y práctica, que unifique el entretenimiento y la organización personal en una única herramienta.

Inteligencia Artificial Usada

Para asistir en el proceso de diseño y desarrollo de la aplicación se empleó Gemini 2.5 Pro, un modelo avanzado de inteligencia artificial desarrollado por Google. Esta herramienta se utilizó como asistente de programación y apoyo creativo, permitiendo generar código, optimizar estructuras y proponer soluciones de diseño de manera eficiente.

Entre sus principales beneficios destacan su capacidad para comprender instrucciones complejas, su adaptabilidad al contexto del proyecto, y su habilidad para generar código limpio y funcional en distintos lenguajes, incluyendo HTML5, CSS y JavaScript vanilla. Gracias a estas características, Gemini 2.5 Pro se convirtió en una herramienta fundamental para agilizar el desarrollo, reducir errores y mejorar la calidad final de la interfaz.

Prompt Inicial

Actúa como un desarrollador web con amplia experiencia en el diseño y desarrollo de aplicaciones e interfaces, aplicando buenas prácticas de programación, accesibilidad, usabilidad y diseño visual.

Desarrolla una aplicación web de reproductor multimedia local con integración de calendario, utilizando HTML5, CSS y JavaScript puro (vanilla JavaScript). No utilices frameworks ni librerías externas, ni para JavaScript ni para CSS.

La aplicación debe incluir las siguientes características:

  • Reproducción de archivos multimedia locales (audio y video).

  • Controles básicos de reproducción: reproducir, pausar, detener, avanzar, retroceder y ajustar el volumen.

  • Lista de reproducción dinámica, con opciones para agregar, eliminar y reordenar archivos.

  • Integración con un calendario, que permita crear eventos o recordatorios y vincularlos con archivos multimedia.

  • Temas personalizables, incluyendo modo oscuro.

  • Persistencia de datos mediante almacenamiento local, conservando listas de reproducción y eventos entre sesiones.

  • Diseño responsive, adaptable a distintos tamaños de pantalla.

  • Interfaz accesible y usable, con elementos bien contrastados, etiquetas semánticas y navegación intuitiva.

Asegúrate de:

  • Mantener un código limpio, modular y semántico, facilitando su comprensión y mantenimiento.

  • Respetar una estructura de proyecto clara, separando correctamente la lógica (JavaScript), el estilo (CSS) y el contenido (HTML).

  • Comentar el código de forma adecuada para explicar su funcionamiento.

Resultado:

La estética de la aplicación debe de ser mas amigable, las esquinas deben de ser mas redondeadas y los colores más llamativos
